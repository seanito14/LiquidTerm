# Accessibility

## Core Features
1. **Font Scaling**: Global slider (12–28pt) + per-pane override.
2. **High Contrast**: Toggle in settings; ensures WCAG 2.1 AA compliance.
3. **Reduce Transparency**: Disables blur; replaces with solid background.
4. **VoiceOver**: Labels for panes/tabs (e.g., "Pane 1, Shell, Active").
5. **Keyboard Navigation**: Full control via shortcuts (see [Keyboard Shortcuts](#)).

## Dyslexia Support
- Optional monospace font with increased letter spacing.
- User can upload custom fonts.

## Testing
- Audit with VoiceOver and Accessibility Inspector.
- Test with "Reduce Transparency" and "Increase Contrast" enabled.
